# Work-Time 앱 기획 개요

> **컨셉**: 우리 코딩 스터디 전용 출석 관리 앱
> **톤앤매너**: 가볍고 부담 없는, 스터디원끼리 쓰는 내부 툴 느낌
> **고도화 방향**: 반응이 좋으면 워크스페이스 기반으로 확장

---

## MVP 범위

**단일 스터디 그룹** — 워크스페이스 개념 없이, 앱 = 우리 스터디

| 기능 | 설명 |
|------|------|
| 출석 체크 | 체크인/체크아웃으로 공부 시간 기록 |
| 실시간 현황 | 지금 누가 공부 중인지 실시간 확인 |
| 출석 기록 | 캘린더로 내 출석 기록 + 통계 |
| 멤버 현황 | 스터디원들의 출석 현황 공유 |
| 휴가 | 2시간 단위로 쪼개서 사용 (2h/4h/6h/8h) |
| 지각 판정 · 지각비 | 스터디 시작 시간 기준 지각 자동 판정 + 지각비 누적 |
| 회비 | 월 회비 납부 상태 추적 (납부/미납) |

## 기수별 규칙 (코드 상수 관리)

> MVP에서는 설정 UI 없이 코드 상수로 관리. 기수 변경 시 개발자가 직접 수정.
> 고도화 시 관리자 설정 화면으로 이관.

| 규칙 | 예시 | 비고 |
|------|------|------|
| 스터디 시작 시간 | 10:00 | 이 시간 이후 체크인 → 지각 |
| 지각비 | 1,000원/회 | 지각 1회당 금액 |
| 월 회비 | 10,000원 | 매월 납부 금액 |
| 기수 | 5기 | 현재 기수 표시용 |

## 화면 구조 (3탭)

```
┌──────────┬──────────┬──────────┐
│   대시보드  │   기록    │   멤버    │
└──────────┴──────────┴──────────┘
```

| 탭 | 화면 | 핵심 |
|----|------|------|
| 대시보드 (Dashboard) | 메인 화면 | 체크인/아웃, 타이머, 휴가, 지각 판정, 공부 중인 멤버 |
| 기록 (History) | 내 출석 기록 | 캘린더 (출석/지각/휴가 표시), 일별 상세, 월간 통계 + 지각비 |
| 멤버 (Members) | 스터디원 현황 | 멤버 목록, 회비·지각비 현황, 주간/월간 랭킹 |
