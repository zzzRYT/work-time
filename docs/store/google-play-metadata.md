## 긴 설명

> 스터디 그룹의 출석과 공부 시간을 손쉽게 관리하세요!
>
> WorkTime은 스터디 그룹 멤버들의 출석 체크와 공부 시간을 추적하고 관리할 수 있는 앱입니다. 간편한 체크인/체크아웃 시스템으로 출석을 기록하고, 실시간으로 공부 시간을 확인할 수 있습니다.
>
> 주요 기능:
>
> - 출석 체크인/체크아웃: 원터치로 출석을 기록하세요. 체크인 시 공부 시간이 자동으로 측정되며, 체크아웃 시 기록이 저장됩니다.
>
> - 실시간 공부 시간 추적: 현재 공부 중인 시간을 실시간으로 확인할 수 있습니다. 오늘의 누적 공부 시간도 한눈에 볼 수 있습니다.
>
> - 월간 출석 캘린더: 캘린더 형태로 한 달간의 출석 현황을 확인하세요. 날짜별 출석 여부와 공부 시간을 한눈에 파악할 수 있습니다.
>
> - 휴가 관리: 사전에 휴가를 등록하여 출석 관리에 반영할 수 있습니다. 그룹 내 휴가 현황도 함께 확인할 수 있습니다.
>
> - 주간/월간 랭킹: 스터디 그룹 내에서 누가 가장 열심히 공부했는지 랭킹으로 확인하세요. 주간 및 월간 기준으로 공부 시간 순위를 제공합니다.
>
> - 회비 납부 관리: 스터디 그룹의 회비 납부 현황을 관리할 수 있습니다. 납부 여부를 기록하고 한눈에 확인하세요.
>
> WorkTime으로 스터디 그룹을 더 효율적으로 운영하세요!

## 스크린샷 준비 체크리스트

Google Play 등록에 필요한 스크린샷 목록입니다. 최소 2장, 최대 8장 등록 가능합니다.

### 필수 스크린샷

- [ ] **메인 화면 (체크인 전)** - 체크인 버튼이 보이는 메인 대시보드
- [ ] **메인 화면 (체크인 후)** - 실시간 타이머와 체크아웃 버튼이 보이는 상태
- [ ] **출석 캘린더** - 월간 출석 현황이 표시된 캘린더 화면
- [ ] **랭킹 화면** - 주간 또는 월간 공부 시간 랭킹

### 권장 스크린샷

- [ ] **출석 히스토리** - 날짜별 체크인/체크아웃 기록 목록
- [ ] **휴가 관리** - 휴가 등록 또는 휴가 현황 화면
- [ ] **회비 납부 현황** - 멤버별 회비 납부 상태 화면
- [ ] **멤버 관리** - 스터디 그룹 멤버 목록 화면

| 항목 | 요구사항 |

| 형식 | JPEG 또는 24비트 PNG (알파 없음) |
| 최소 크기 | 320px |

| 가로세로 비율 | 최대 2:1 |
