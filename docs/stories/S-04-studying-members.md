# S-04. 지금 공부 중인 멤버 확인하기

> **탭**: Dashboard
> **관련 스펙**: REQ-1.6.1 ~ REQ-1.6.5
> **상태**: 🟡 부분 구현

---

## 유저 스토리

**AS** 스터디원
**I WANT TO** 지금 누가 공부하고 있는지 실시간으로 확인
**SO THAT** 같이 공부하는 동기부여를 받을 수 있다

---

## 플로우

### 플로우 1: 공부 중인 멤버가 있을 때

```
[대시보드 스크롤 다운] → ["지금 공부 중 (3명)" 섹션]
→ [멤버별 프로필 + 이름 + 경과 시간 목록]
```

| 단계 | 사용자 행동 | 시스템 반응 | UI 위치 |
|------|------------|-----------|---------|
| 1 | 대시보드에서 아래로 스크롤 | "지금 공부 중" 섹션 노출 | 출석 카드 아래 섹션 |
| 2 | — | 섹션 타이틀 + 인원수 표시 (e.g. "지금 공부 중 3명") | 섹션 헤더 |
| 3 | — | 체크인 중인 멤버 목록 표시 | 섹션 본문 리스트 |
| 4 | — | 각 멤버: 프로필 색상 아바타 + 이름 + 경과 시간 | 각 멤버 행 |
| 5 | — | 경과 시간 실시간 갱신 (polling 30s) | 각 멤버 행 우측 |

### 플로우 2: 아무도 공부하지 않을 때

```
[대시보드] → ["지금 공부 중 0명"] → ["아직 아무도 시작 안 했어요" 안내]
```

| 단계 | 사용자 행동 | 시스템 반응 | UI 위치 |
|------|------------|-----------|---------|
| 1 | 대시보드 진입 | "지금 공부 중 0명" 표시 | 섹션 헤더 |
| 2 | — | "아직 아무도 시작 안 했어요" 빈 상태 안내 | 섹션 본문 |

```
┌─────────────────────────────────┐

│  ┌─────────────────────────┐    │
│  │ 🟣 홍길동    02:15:30    │    │  ← 멤버 행 (아바타 + 이름 + 시간)

│  │ 🔵 김철수    01:45:12    │    │

│  └─────────────────────────┘    │
└─────────────────────────────────┘
```

**빈 상태:**
```
┌─────────────────────────────────┐
│  지금 공부 중  0명                │
│                                 │
│       아직 아무도 시작 안 했어요     │  ← 회색 텍스트, 가운데 정렬
│                                 │
└─────────────────────────────────┘
```

---

## 데이터 흐름

```
members query (polling 30s)
    → currentStatus == "STUDYING" 인 멤버 필터링
    → 각 멤버의 activeSession.checkInTime 기준 경과 시간 계산
    → 경과 시간 순 정렬 (오래 공부한 순)
```

## 서브 이슈

### S-04-1. 빈 상태 UI

> **우선순위**: 중간 | **상태**: 🔴 미구현

아무도 공부하지 않을 때 안내 메시지와 아이콘을 표시한다.

**동작:**
| 조건 | 표시 |
|------|------|
| 공부 중인 멤버 0명 | 아이콘 + "아직 아무도 시작 안 했어요" 텍스트 |
| 공부 중인 멤버 1명 이상 | 멤버 목록 (기존 동작) |

**UI:**
```
┌─────────────────────────────────┐
│  지금 공부 중  0명                │
│                                 │
│           📚                    │  ← 아이콘 (책/연필 등)
│   아직 아무도 시작 안 했어요        │  ← 회색 텍스트, 가운데 정렬
│                                 │
└─────────────────────────────────┘
```

**구현 포인트:**
- `studyingMembers.length === 0` 일 때 조건부 렌더링
- 회색 텍스트 + 심플 아이콘 (과하지 않게)

---

### S-04-2. 멤버 수 카운트 배지

> **우선순위**: 낮음 | **상태**: 🔴 미구현

섹션 타이틀에 현재 공부 중인 인원수를 표시한다.

**동작:**
| 인원 | 타이틀 표시 |
|------|-----------|
| 0명 | "지금 공부 중  0명" |
| 3명 | "지금 공부 중  3명" |

**스타일:**
- "지금 공부 중" → 볼드 텍스트
- "3명" → 메인 색상(Primary) 또는 배지 형태

**구현 포인트:**
- `studyingMembers.length`를 타이틀 옆에 렌더링
- 숫자 부분만 색상 강조 또는 원형 배지

---

### S-04-3. 멤버 탭 → 오늘 출석 상세

> **우선순위**: 낮음 | **상태**: 🔴 미구현

공부 중인 멤버 행을 탭하면 해당 멤버의 오늘 출석 상세를 바텀시트로 보여준다.

**동작:**
| 단계 | 사용자 행동 | 시스템 반응 | UI 위치 |
|------|------------|-----------|---------|
| 1 | 멤버 행 탭 | `dayDetail(memberId, today)` query 호출 | 멤버 행 |
| 2 | — | 바텀시트: 해당 멤버의 오늘 세션 목록 표시 | 바텀시트 |

**바텀시트 내용:**
```
┌─────────────────────────────────┐
│  🟣 홍길동의 오늘 기록              │
│                                 │
│  09:55 → 12:30  (2h 35m)       │  ← 세션 1
│  13:00 → 진행 중  (1h 30m~)     │  ← 세션 2 (현재)
│                                 │
│  오늘 총: 4시간 5분               │
└─────────────────────────────────┘
```

**구현 포인트:**
- 멤버 행에 `onPress` 핸들러 추가
- 바텀시트 컴포넌트 재활용 (`@gorhom/bottom-sheet`)
- S-06 (일별 상세) 컴포넌트를 바텀시트 안에서 재사용 가능
