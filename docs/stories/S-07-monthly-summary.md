## 플로우

### 플로우 1: 월간 통계 조회

```
[기록 탭 진입] → [캘린더 아래 월간 요약 카드]
→ [출석일 / 총 공부시간 / 평균 / 지각 / 휴가 / 지각비 확인]
```

| 단계 | 사용자 행동 | 시스템 반응 | UI 위치 |
|------|------------|-----------|---------|
| 1 | 기록 탭 진입 또는 스크롤 | 월간 요약 카드 표시 | 캘린더 + 일별 상세 아래 |
| 2 | — | 6개 통계 항목 렌더링 | 요약 카드 내부 (그리드) |

### 플로우 2: 월 변경 시 통계 갱신

```
[캘린더 스와이프 → 2월] → [2월 통계로 갱신]
```

| 단계 | 사용자 행동 | 시스템 반응 | UI 위치 |
|------|------------|-----------|---------|
| 1 | 캘린더 월 이동 | `monthlySummary` query 재호출 | — |
| 2 | — | 해당 월 통계로 갱신 | 월간 요약 카드 |

## UI 레이아웃

캘린더(잔디) **상단**에 작은 인라인 요약 바로 배치한다. 핵심 3개 항목만 한 줄에 표시.

```
┌─────────────────────────────────┐
│       ◀  2026년 3월  ▶          │  ← 월 헤더
│                                 │
│  📚 68h 30m  ·  📅 15일  ·  💸 3,000원 │  ← 요약 바 (한 줄)
│                                 │
│  ⬜  🟢  🟢  🟡  🟢  🟢  ⬜    │  ← 잔디 캘린더 (S-05)
│  ⬜  🟢  🔵  🟢  🟢  🔴  ⬜    │
│  ...                            │
└─────────────────────────────────┘
```

**요약 바 항목:**
| 아이콘 | 항목 | 표시 예시 |
|-------|------|---------|
| 📚 | 총 공부 시간 | 68h 30m |
| 📅 | 공부 일수 | 15일 |
| 💸 | 총 지각비 | 3,000원 |

---

## 통계 항목 상세

| 항목 | 필드 | 계산 방법 | 표시 형식 |
|------|------|----------|----------|
| 출석일 | attendanceDays | 체크인 기록이 있는 날 카운트 | `N일` |
| 총 공부 시간 | totalStudyMinutes | 전체 세션 durationMinutes 합산 | `Nh Mm` |
| 일 평균 | averageDailyMinutes | totalStudyMinutes / attendanceDays | `Nh Mm` |
| 지각 횟수 | lateCount | isLate == true 세션 카운트 | `N회` |
| 휴가 일수 | vacationDays | DailyVacation 기록 카운트 | `N일` |
| 누적 지각비 | totalLateFee | lateCount × lateFeeAmount | `N,NNN원` |

---

## 데이터 흐름

```
monthlySummary(memberId, year, month)
    → { attendanceDays, totalStudyMinutes, averageDailyMinutes,
        lateCount, vacationDays, totalLateFee }

* 캘린더 월 변경 시 year/month 파라미터 변경 → 자동 refetch
```

## 서브 이슈

### S-07-1. 월간 요약 바 렌더링

> **우선순위**: 높음 | **상태**: 🔴 미구현

캘린더 상단에 핵심 3개 통계를 한 줄 요약 바로 표시한다.

**표시 항목:**
| 항목 | 필드 | 포맷 | 아이콘 |
|------|------|------|-------|
| 총 공부 시간 | totalStudyMinutes | `Nh Mm` | 📚 |
| 공부 일수 | attendanceDays | `N일` | 📅 |
| 총 지각비 | totalLateFee | `N,NNN원` | 💸 |

**UI:**
```
│  📚 68h 30m  ·  📅 15일  ·  💸 3,000원 │
```

**스타일:**
- 한 줄 가로 배치, 가운데 정렬
- 항목 사이 `·` 구분자
- 작은 텍스트 (14px 정도)
- 배경: 연회색 또는 투명

**위치:** 월 헤더와 잔디 캘린더 사이

**구현 포인트:**
- `monthlySummary` query 결과에서 3개 필드 추출
- flexDirection: row, justifyContent: space-around

---

### S-07-2. 항목별 포맷팅

> **우선순위**: 높음 | **상태**: 🔴 미구현

각 통계 항목의 숫자를 사용자 친화적인 형식으로 포맷한다.

**포맷 규칙:**
| 항목 | 원본 | 포맷 결과 | 비고 |
|------|------|---------|------|
| 총 공부 시간 | 4110 (분) | "68h 30m" | 분 → 시간+분 변환 |
| 공부 일수 | 15 | "15일" | 숫자 + "일" |
| 총 지각비 | 3000 | "3,000원" | 천 단위 콤마 + "원" |
| 지각비 0 | 0 | "0원" | 0도 정상 표시 |

**유틸 함수:**
```typescript
formatStudyTime(minutes: number): string  // 4110 → "68h 30m"
formatCurrency(amount: number): string    // 3000 → "3,000원"
```

**구현 포인트:**
- `shared/lib/format.ts`에 포맷 유틸 함수 생성
- `toLocaleString('ko-KR')` 활용하여 콤마 포맷

---

### S-07-3. 월 변경 연동

> **우선순위**: 높음 | **상태**: 🔴 미구현

캘린더 스와이프(S-05-4) 또는 화살표(S-05-6)로 월 이동 시 요약 바 데이터를 자동 갱신한다.

**동작:**
| 시점 | 동작 |
|------|------|
| 월 이동 | `monthlySummary(memberId, newYear, newMonth)` refetch |
| 데이터 로딩 중 | 이전 값 유지 또는 스켈레톤 표시 |
| 로딩 완료 | 요약 바 값 갱신 |

**구현 포인트:**
- S-05의 `year`, `month` 상태를 공유하여 `monthlySummary` query 파라미터로 전달
- `useQuery`의 `variables` 변경 → 자동 refetch

---

### S-07-4. 빈 월 통계

> **우선순위**: 중간 | **상태**: 🔴 미구현

기록이 없는 달에는 모든 값을 0 또는 "-"로 표시한다.

**동작:**
| 항목 | 기록 없을 때 표시 |
|------|----------------|
| 총 공부 시간 | "0h 0m" |
| 공부 일수 | "0일" |
| 총 지각비 | "0원" |

**UI:** 요약 바 구조 유지, 값만 0으로 표시 (빈 잔디 캘린더 S-05-8과 일관)

**구현 포인트:**
- `monthlySummary` 응답이 모두 0일 때도 정상 렌더링
- null/undefined 방어 코드
